## 第5步：体验AI功能（10分钟）

### 打开一个代码文件

1. **左侧文件树中**，找到 `duomotai/index.html`
2. **双击打开**
3. **看到代码显示在编辑区**

### 方式1：使用快捷键 Ctrl+K（推荐）

这是Cursor最强大的功能——**AI对话框**：

1. **在代码编辑区，按 `Ctrl+K`**
2. **屏幕右侧会出现AI对话框**：
   ```
   ┌─────────────────────────┐
   │  Cursor AI              │
   │                         │
   │  [输入框 - 说你的需求]  │
   │                         │
   │  [↑] 发送按钮           │
   └─────────────────────────┘
   ```

3. **输入你的需求**，比如：
   ```
   "这个HTML文件的主要功能是什么？"
   ```

4. **按 Enter 或点击发送**

5. **AI会分析代码并回答**：
   ```
   右侧面板显示：
   "这个文件是多魔汰辩论系统的主页面，包含以下功能：
   1. 用户登录和认证
   2. 话题输入框

   4. 辩论启动按钮
   ```

也可以用传统方式：

1. **右键点击代码**
2. **选择 "Ask Cursor"** 或 **"Generate"**

## 第8步：常见问题解答

### Q1: Cursor会占用多少硬盘空间？
**A**: 约500MB（和VS Code差不多）

### Q2: 我可以同时装Cursor和VS Code吗？
**A**: **完全可以！**它们互不冲突。一些开发者就是两个都装。

### Q3: 免费500次是什么意思？
**A**: 每月你可以用Cursor的AI功能500次。一个"次"大约是：
- 一次Ctrl+K对话 = 1次
- 一次AI代码补全 = 1次

500次/月对于个人开发者足够了。

### Q4: 如果超过了500次怎么办？
**A**: 超过后，你可以：
- 升级到Cursor Pro（$20/月，无限次）
- 或者等待下个月配额重置
- 或者继续用VS Code作为补充

### Q5: 我的代码会被Cursor上传吗？
**A**: **你的项目代码不会被保存**。Cursor只在本地读取代码，用于AI理解上下文，不会上传服务器。

### Q6: 如何关闭AI功能，用传统的代码补全？
**A**: 按 `Esc` 可以关闭AI建议，或在设置中禁用。
